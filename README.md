# 鳥人間 機体設計シミュレーター 3D

人力飛行機(鳥人間コンテスト機)の機体を設計し、3Dでテストフライトできるブラウザシミュレーターです。
追加のインストールは不要で、HTMLファイルを開くだけで動作します。

## 特徴

- **🛠 設計モード** — 主翼・尾翼・桁・プロペラ・パイロットなどを細かく設定
  - 翼型(DAE-31 / FX 76-MP / SD-7037 / E-387 など)の選択
  - 平面形(矩形翼・テーパー翼・クレセント翼・超大型機など)のテンプレート
  - リブ間隔、プランク範囲、エルロン、上反角(ジグ/フラット)などの構造設定
- **🛫 テストフライトモード** — 設計した機体を3D空間で飛ばして挙動を確認
  - パイロット視点(FPV)・HUD計器・高度/速度グラフ
  - 風・突風・気温・サーマルなどの飛行条件を変更可能
- **性能評価** — 滑空性能・出力余裕・安定性・構造強度・発進容易さを自動スコアリング

## 動かし方

1. このリポジトリをダウンロード(またはクローン)します。
2. `birdman-3d_37.html` を Web ブラウザで開きます。

インターネット接続があれば、CDN から読み込む [three.js](https://threejs.org/) (r128) を使って 3D 描画が行われます。

## 操作方法

| 操作 | 動作 |
| --- | --- |
| ドラッグ | 視点の回転 |
| ホイール | ズーム |
| Shift + ドラッグ | 平行移動 |

テストフライト中は画面上の方向パッド(▲▼◀▶)とラダー(⟲⟳)で機体を操縦できます。

## 技術

- 単一の HTML ファイル(HTML / CSS / JavaScript)
- 3D 描画: [three.js](https://threejs.org/) r128(CDN 読み込み)
